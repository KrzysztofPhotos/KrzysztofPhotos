## 🚀 Featured Projects

### 📡 OLX Sniper & Management Panel
An automated, 24/7 web scraping and management tool built to find the best deals.
- **Tech Stack:** Python, Flask, SQLite, BeautifulSoup4, systemd
- **Features:** Continuously monitors OLX, detects price-range deals, sends real-time Telegram notifications, and features a responsive web-based management panel. Designed for headless operation on a Raspberry Pi.

### ⏱️ Kitchen Timer (FRDM-KL05Z)
An embedded systems project demonstrating low-level hardware control.
- **Tech Stack:** C (CMSIS register-level), NXP MKL05Z4 Microcontroller
- **Features:** GPIO interrupts, PWM audio generation for alarms, PIT timer for precise countdowns, and an LCD 1602 display.

### 🛒 C++ OOP Marketplace System
A console-based application simulating an online marketplace, created for my Object-Oriented Programming course.
- **Tech Stack:** C++17
- **Features:** Object-oriented design, custom user authentication, file I/O for data persistence, colored terminal output, and full admin management functionalities.
